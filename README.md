# GamesDB-Browser
https://thegamesdb.net/

Web based application designed to provide more robust searchability and filtering of the gamesDB.

### Primary Functionality
- __Search by Name__  
- __Browse by Platform__  
- __Filter Search__  
- __Top Rated Lists__   
- __Show Random Game__    

### Applicable Filters
_- Genre_  
_- Date Released_  
_- Publisher_  
_- Developer_  
_- Platform_  
_- Rating_  
_- Players_  
_- Coop_  
_- Community Review_
