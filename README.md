# GamesDB-Browser
https://thegamesdb.net/

Web based application designed to provide more robust searchability and filtering of the gamesDB.

### Primary Functionality
-__Search by Name__  
-__Browse by Platform__  
-__Filter Search__  
-__Top Rated Lists__   
-__Show Random Game__    

### Applicable Filters
_Genre_  
_Date Released_  
_Publisher_  
_Developer_  
_Platform_  
_Rating_  
_Players_  
_Coop_  
_Community Review_
